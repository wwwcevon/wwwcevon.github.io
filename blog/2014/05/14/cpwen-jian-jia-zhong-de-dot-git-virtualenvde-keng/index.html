cp文件夹中的.git .virtualenv的坑

May 14th, 2014

cp 不会 copy 隐藏文件(以.开头的)

如 .git .virtualenv

所以如果有用 virtualenv 包装环境，不要使用 cp 或者 mv 直接复制或者移动

如果非要使用，可以加上-a

注意: 直接移动 virtualenv 后会无法使用，因为虚拟环境是写死绝对路径的

可以在 .virtualenv/bin/activate 中找到并修改
